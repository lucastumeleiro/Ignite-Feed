## Getting started

Clone o projeto e acesse a pasta

```bash
$ git clone https://github.com/lucastumeleiro/Ignite-Feed.git
$ cd
```

Siga os passos:

```bash
# Instale as dependências
$ yarn
# Inicie o projeto
$ yarn start
#O projeto pode ser acessado em: http://localhost:3000
```
